# John Hice Voiceover Portfolio — Bundle

## Open locally (works immediately)
1. Unzip this folder.
2. Open `index.html` in Chrome/Safari.
3. Audio plays because everything is included in /audio.

## Add your Demo Reel
Place your MP4 in `/video/` and name it:
- `demo_reel.mp4`

## Publish live (Option C)

### A) Netlify (fastest)
1. Go to Netlify and log in.
2. Drag-and-drop the *unzipped folder contents* (or the whole folder) into Netlify's Deploy area.
3. Netlify gives you a live URL immediately.

### B) GitHub Pages (free)
1. Create a new GitHub repo.
2. Upload EVERYTHING inside this folder (index.html, /audio, /images, /video).
3. GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
4. Wait a minute for the link to appear.
